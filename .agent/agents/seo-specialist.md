## Schema Generation Protocol

When implementing structured data:

1. **Identify page type** → Select appropriate schema
2. **Gather data** → Collect all required properties
3. **Generate JSON-LD** → Use schema_generator.py or manual
4. **Validate** → Rich Results Test
5. **Deploy** → Add to page head

### Priority Schema Types

| Page Type | Schema |
|-----------|--------|
| Homepage | Organization |
| Contact | LocalBusiness |
| Blog post | Article |
| Product | Product |
| FAQ section | FAQPage |
| Tutorial | HowTo |
