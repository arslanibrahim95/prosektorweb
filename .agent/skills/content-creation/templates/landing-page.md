# Landing Page Template

> Copy and customize this template for landing pages.

---

## Page Metadata

```yaml
title: "[Value Proposition | Brand]"
description: "[150-160 char description with CTA hint]"
og_image: "[social-share-image.jpg]"
canonical: "[full-url]"
```

## Above the Fold

### Hero Section

```
[HEADLINE - Clear Value Proposition]
────────────────────────────────────
Maximum 10 words, benefit-focused

[SUBHEADLINE]
────────────────────────────────────
Expand on headline, 15-25 words
Address the "how" or "for whom"

[HERO IMAGE/VIDEO]
────────────────────────────────────
Product in action or outcome visualization

[PRIMARY CTA BUTTON]
────────────────────────────────────
Action verb + benefit (e.g., "Start Free Trial")

[TRUST INDICATORS]
────────────────────────────────────
"Trusted by 10,000+ companies" | ⭐⭐⭐⭐⭐ 4.9/5 | Logos
```

## Implementation Checklist

### Content
- [ ] Headline clearly states value
- [ ] Single focused goal
- [ ] Benefits > Features
- [ ] Social proof included
- [ ] FAQ addresses objections
- [ ] Clear, repeated CTAs

### Design
- [ ] Above-fold is complete
- [ ] Visual hierarchy clear
- [ ] Consistent brand colors
- [ ] High-quality images
- [ ] Mobile responsive

### Conversion
- [ ] One primary CTA
- [ ] Minimal form fields
- [ ] Trust indicators present
- [ ] No distracting navigation
- [ ] Load time < 3 seconds

### Technical
- [ ] Meta tags optimized
- [ ] OG image set
- [ ] Analytics tracking
- [ ] Form tested
- [ ] Mobile tested
