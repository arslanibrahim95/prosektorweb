# FAQ Section Template

> Template for creating SEO-optimized FAQ sections.

---

## Schema Implementation

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Question text here]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "[Answer text here]"
      }
    }
  ]
}
```

## FAQ Writing Guidelines

### Question Format

| Do | Don't |
|----|-------|
| Use natural language | Use jargon |
| Match search queries | Make up questions |
| Be specific | Be vague |
| Front-load keywords | Bury keywords |

### Answer Format

| Element | Guideline |
|---------|-----------|
| Length | 50-200 words |
| Tone | Helpful, clear |
| Structure | Direct answer first, then details |
| Links | Include where relevant |

## Best Practices

### SEO Optimization

- [ ] Questions match actual search queries
- [ ] Include target keywords naturally
- [ ] Implement FAQPage schema
- [ ] Link to detailed content pages

### UX Considerations

- [ ] Collapsible/accordion format
- [ ] Search functionality (for large FAQs)
- [ ] Categorized sections
- [ ] Mobile-friendly

### Content Quality

- [ ] Answers are accurate and current
- [ ] Complex topics have supporting links
- [ ] Tone matches brand voice
- [ ] Regular review and updates
