# Blog Writing

> Guidelines for creating effective blog content.

---

## 1. Blog Post Structure

### Anatomy of a Blog Post

```
TITLE
  └── Compelling, keyword-optimized (50-60 chars)

INTRODUCTION (150-200 words)
  ├── Hook (attention-grabbing opening)
  ├── Problem statement
  └── Promise (what reader will learn)

BODY (Main content)
  ├── H2: Section 1
  │   ├── Content
  │   └── Supporting elements
  ├── H2: Section 2
  │   ├── Content
  │   └── Supporting elements
  └── H2: Section 3 (+ more as needed)

CONCLUSION (100-150 words)
  ├── Summary of key points
  ├── Call-to-action
  └── Next steps

AUTHOR BIO
  └── Credentials, links
```

---

## 2. Title Formulas

### High-Converting Patterns

| Pattern | Example |
|---------|---------|
| How to [Achieve X] | How to Increase Website Traffic |
| [Number] Ways to [Do X] | 7 Ways to Improve SEO |
| [Number] [Topic] Tips | 10 Content Writing Tips |
| The Ultimate Guide to [X] | The Ultimate Guide to Email Marketing |
| [Topic]: What You Need to Know | Local SEO: What You Need to Know |
| Why [X] Matters | Why Core Web Vitals Matter |
| [X] vs [Y]: Which is Better? | WordPress vs Next.js: Which is Better? |

### Title Checklist
- [ ] Contains target keyword
- [ ] 50-60 characters
- [ ] Creates curiosity
- [ ] Promises value
- [ ] Matches content

## 3. Introduction Techniques

### Hook Types

| Type | Example |
|------|---------|
| **Question** | "Ever wondered why your content isn't ranking?" |
| **Statistic** | "93% of online experiences begin with a search engine." |
| **Story** | "Last year, our traffic dropped 60% overnight..." |
| **Bold statement** | "Most SEO advice is outdated—here's what works now." |
| **Problem** | "You're creating content, but no one's reading it." |

### Introduction Structure
```
[Hook - 1-2 sentences]
[Problem/Context - 2-3 sentences]
[Promise - 1 sentence]
[Transition to content]
```

---

## 4. Body Content

### Section Structure

Each H2 section should:
- Address one main point
- Be 200-400 words
- Include supporting elements
- Flow to next section

### Supporting Elements

| Element | When to Use |
|---------|-------------|
| **Lists** | Multiple related items |
| **Tables** | Comparisons, data |
| **Images** | Visual concepts |
| **Code blocks** | Technical content |
| **Quotes** | Expert opinions |
| **Examples** | Concrete applications |

### Transition Phrases

- "Now that we've covered X, let's look at..."
- "With that foundation, here's..."
- "But that's not all..."
- "Next, we'll explore..."

---

## 5. Conclusion

### Effective Endings

| Component | Purpose |
|-----------|---------|
| **Summary** | Reinforce key takeaways |
| **CTA** | Guide next action |
| **Question** | Encourage engagement |
| **Future hook** | Tease related content |

### CTA Options

- "Ready to get started? [Contact us]"
- "Download our free template"
- "Subscribe for more tips"
- "Share your experience in the comments"
- "Read our related guide on [topic]"

---

## 6. Content Length

### By Type

| Content Type | Recommended Length |
|--------------|-------------------|
| News/Updates | 500-800 words |
| How-to | 1,000-1,500 words |
| Guides | 1,500-3,000 words |
| Pillar content | 3,000-5,000+ words |

### Length Guidelines

- **Long enough** to cover topic thoroughly
- **Short enough** to maintain engagement
- **Focused** on value, not word count
- **Scannable** regardless of length

## 7. Blog Post Checklist

### Pre-Writing
- [ ] Keyword research done
- [ ] Search intent understood
- [ ] Outline created
- [ ] Resources gathered

### Writing
- [ ] Compelling title
- [ ] Strong hook
- [ ] Clear structure
- [ ] Value delivered
- [ ] Examples included

### Editing
- [ ] Grammar/spelling checked
- [ ] Readability improved
- [ ] Formatting consistent
- [ ] Links working

### SEO
- [ ] Title tag optimized
- [ ] Meta description written
- [ ] Headers contain keywords
- [ ] Images have alt text
- [ ] Internal links added

---

> **Rule:** Every blog post should answer: "What will the reader learn, and why should they care?"
