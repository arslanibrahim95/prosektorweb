# Landing Pages

> Anatomy and best practices for high-converting landing pages.

---

## 1. Landing Page Structure

### Above the Fold

```
HEADLINE
  └── Clear, benefit-focused value proposition

SUBHEADLINE
  └── Support headline, expand promise

HERO IMAGE/VIDEO
  └── Visual representation of offering

PRIMARY CTA
  └── Single, prominent action button

TRUST INDICATORS
  └── Logos, ratings, security badges
```

### Below the Fold

```
PROBLEM/PAIN POINTS
  └── Identify what visitor struggles with

SOLUTION
  └── How you solve the problem

FEATURES/BENEFITS
  └── What they get (benefits > features)

SOCIAL PROOF
  └── Testimonials, case studies, numbers

FAQ/OBJECTIONS
  └── Address common concerns

FINAL CTA
  └── Repeat call-to-action
```

## 2. Headline Formulas

### Effective Patterns

| Pattern | Example |
|---------|---------|
| [Achieve X] Without [Pain Y] | "Grow Traffic Without Technical SEO Knowledge" |
| The [Adjective] Way to [Benefit] | "The Simple Way to Manage Your Compliance" |
| [Get X] in [Time] | "Get More Leads in 30 Days" |
| Stop [Pain]. Start [Benefit] | "Stop Wasting Time. Start Growing." |
| [Number] [People] Use [Product] | "10,000+ Businesses Use Our Platform" |

### Headline Checklist
- [ ] Clear benefit stated
- [ ] Speaks to target audience
- [ ] Creates urgency or curiosity
- [ ] Under 10 words ideal
- [ ] No jargon

## 3. Value Proposition

### Components

| Element | Purpose |
|---------|---------|
| **Target** | Who is this for? |
| **Problem** | What pain do they have? |
| **Solution** | What do you offer? |
| **Differentiation** | Why you vs competitors? |
| **Proof** | Why believe you? |

### Formula
```
For [TARGET AUDIENCE] who [HAVE PROBLEM],
[PRODUCT] is a [CATEGORY]
that [KEY BENEFIT].
Unlike [COMPETITOR/ALTERNATIVE],
our product [KEY DIFFERENTIATOR].
```

---

## 4. Features vs Benefits

### Conversion Rule

> **Features tell, benefits sell.**

| Feature (What it is) | Benefit (What it does for them) |
|---------------------|--------------------------------|
| 24/7 Support | Never wait for help |
| Cloud-based | Access anywhere |
| 256-bit encryption | Your data stays safe |
| AI-powered | Save hours every week |

### Writing Benefits

1. Start with feature
2. Ask "So what?"
3. Answer = benefit
4. Make it emotional

## 5. Social Proof

### Types

| Type | When to Use |
|------|-------------|
| **Testimonials** | Customer quotes |
| **Case studies** | Detailed success stories |
| **Numbers** | User count, results |
| **Logos** | Recognizable clients |
| **Reviews** | Third-party validation |
| **Awards** | Industry recognition |

### Effective Testimonials

Include:
- Full name + photo
- Company/role
- Specific result
- Before/after if possible

```
"We increased leads by 300% in 3 months."
— Name, Title, Company
```

## 6. Call-to-Action (CTA)

### Button Best Practices

| Do | Don't |
|----|-------|
| Use action verbs | "Submit" |
| Be specific | "Click here" |
| Create urgency | "Learn more" |
| Show value | Generic phrases |

### CTA Examples

| Weak | Strong |
|------|--------|
| Submit | Get My Free Guide |
| Buy Now | Start Growing Today |
| Sign Up | Create Free Account |
| Download | Download Your Template |

### CTA Placement

- Above the fold (primary)
- After benefits section
- After testimonials
- Bottom of page (final)

## 7. Form Optimization

### Field Guidelines

| Rule | Reason |
|------|--------|
| Fewer fields = higher conversion | Reduce friction |
| Name + Email minimum | Most essential |
| Phone only if necessary | Deters signups |
| Required fields marked | Clear expectations |

### Form Placement

- Right side of fold
- Near CTAs
- After building value

---

## 8. Mobile Optimization

### Mobile-First Rules

- [ ] Single column layout
- [ ] Large touch targets (48px+)
- [ ] Readable text (16px+)
- [ ] Fast load time (<3s)
- [ ] Sticky CTA button
- [ ] Compressed images

## 9. Landing Page Checklist

### Structure
- [ ] Clear headline above fold
- [ ] Single focused goal
- [ ] Logical content flow
- [ ] Multiple CTAs

### Content
- [ ] Benefit-focused copy
- [ ] Social proof included
- [ ] FAQ/objections addressed
- [ ] No navigation distractions

### Design
- [ ] Mobile responsive
- [ ] Fast loading
- [ ] Clear visual hierarchy
- [ ] Consistent branding

### Conversion
- [ ] Single goal
- [ ] Clear CTA
- [ ] Minimal form fields
- [ ] Trust indicators

---

> **Rule:** One page, one goal, one CTA. Remove everything that doesn't support conversion.
