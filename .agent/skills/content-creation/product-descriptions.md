# Product Descriptions

> Writing e-commerce product content that converts.

---

## 1. Product Description Structure

### Essential Elements

```
PRODUCT TITLE
  └── Clear, searchable, benefit-hint

SHORT DESCRIPTION (50-100 words)
  └── Quick overview, key benefit

LONG DESCRIPTION (200-500 words)
  └── Full story, features, benefits

SPECIFICATIONS
  └── Technical details, dimensions

SOCIAL PROOF
  └── Reviews, ratings
```

---

## 2. Writing Formula

### AIDA for Products

| Stage | Content |
|-------|---------|
| **Attention** | Headline that stops scrolling |
| **Interest** | Unique features, problem it solves |
| **Desire** | Benefits, emotional appeal |
| **Action** | Clear purchase CTA |

### Benefit-Feature-Proof

```
[BENEFIT] - What it does for them
Because [FEATURE] - What makes it possible
As shown by [PROOF] - Why they should believe
```

Example:
> "Save 2 hours daily (benefit) with our AI-powered automation (feature), just like 10,000+ users already do (proof)."

## 3. Title Optimization

### Elements

| Component | Purpose |
|-----------|---------|
| Brand | Recognition |
| Product name | Identification |
| Key feature | Differentiation |
| Size/variant | Specification |

### Examples

```
[Brand] [Product] - [Key Feature] [Size/Color]

Nike Air Max 90 - Breathable Running Shoes - Men's Black
Apple MacBook Pro 16" - M3 Pro Chip - 18GB RAM
```

### Title Length
- 50-75 characters ideal
- Front-load keywords
- Include key differentiator

## 4. Short Description

### Purpose
- Scannable overview
- Key selling points
- Search snippet preview

### Format
```
[One-line benefit statement]

• Key feature 1
• Key feature 2
• Key feature 3
```

### Example
```
Transform your workspace with the ErgoMax Standing Desk.

• Adjustable height 28-48 inches
• Whisper-quiet motor
• Memory presets for quick switching
• Supports up to 300 lbs
```

---

## 5. Long Description

### Structure

```
OPENING HOOK (1-2 sentences)
  └── Emotional or problem-focused

PROBLEM/NEED (2-3 sentences)
  └── What pain point this solves

SOLUTION STORY (3-4 sentences)
  └── How product addresses need

FEATURE BREAKDOWN
  └── Features with benefits

IDEAL CUSTOMER
  └── Who this is perfect for

CLOSING + CTA
  └── Summary, urgency, action
```

### Example Structure

```markdown
**Finally, a standing desk that keeps up with you.**

Tired of feeling stiff after long work hours? Studies show sitting
8+ hours daily increases health risks by 40%.

The ErgoMax Standing Desk was designed by ergonomics experts to
seamlessly integrate movement into your workday.

**Features:**
- **Whisper-quiet motor** – Change positions without disturbing
  video calls
- **4 memory presets** – Your perfect heights, one button away
- **Cable management** – Built-in trays keep your space clean
- **300 lb capacity** – Handles multiple monitors with ease

**Perfect for:** Remote workers, gamers, creatives, and anyone
who values their health.

**Order now** and transform how you work.
```

## 6. Sensory Language

### Make Products Tangible

| Sense | Words |
|-------|-------|
| **Touch** | Smooth, soft, lightweight, sturdy |
| **Sight** | Sleek, vibrant, compact, elegant |
| **Sound** | Quiet, crisp, silent |
| **Smell** | Fresh, natural, clean |
| **Taste** | Rich, smooth, bold |

### Power Words

| Category | Words |
|----------|-------|
| **Quality** | Premium, handcrafted, professional |
| **Time** | Instant, quick, effortless |
| **Exclusivity** | Limited, rare, exclusive |
| **Trust** | Guaranteed, certified, proven |

## 7. Technical Specifications

### Format

```
SPECIFICATIONS
────────────────────────
Dimensions:    12" x 8" x 2"
Weight:        2.5 lbs
Material:      Aircraft-grade aluminum
Color:         Matte Black
Compatibility: USB-C, Thunderbolt 3/4
Warranty:      2 years
```

### Best Practices

- Use clear labels
- Include units
- Provide comparisons when helpful
- Highlight differentiators

## 8. SEO for Products

### On-Page Elements

| Element | Optimization |
|---------|--------------|
| Title | Primary keyword + brand |
| URL | Short, keyword-rich |
| Description | Keywords naturally placed |
| Images | Descriptive alt text |
| Schema | Product schema markup |

### Keywords

Place primary keyword in:
- Product title
- First 100 words
- One H2 heading
- Image alt text
- Meta description
