# Review Criteria

> Detailed definitions for content quality assessment.

---

## 1. Accuracy (Weight: 20%)

### Definition
Factual correctness, verifiable claims, and truthfulness of content.

### Scoring Guide

| Score | Criteria |
|-------|----------|
| 90-100 | All facts verified, sources cited, no errors |
| 80-89 | Minor factual ambiguities, mostly accurate |
| 70-79 | Some claims need verification |
| 60-69 | Multiple unverified claims |
| < 60 | Factual errors present |

### Checklist

- [ ] Statistics have sources
- [ ] Claims are verifiable
- [ ] No outdated information
- [ ] Technical details are correct
- [ ] Names, dates, numbers accurate

---

## 2. Clarity (Weight: 20%)

### Definition
Readability, comprehension ease, and logical flow.

### Scoring Guide

| Score | Criteria |
|-------|----------|
| 90-100 | Crystal clear, any reader understands |
| 80-89 | Clear with minor complexity |
| 70-79 | Mostly clear, some dense sections |
| 60-69 | Confusing in parts |
| < 60 | Difficult to understand |

### Checklist

- [ ] Sentences under 25 words average
- [ ] Paragraphs 2-4 sentences
- [ ] Jargon explained or avoided
- [ ] Logical progression
- [ ] Clear topic sentences

### Readability Targets

| Audience | Flesch-Kincaid Grade |
|----------|---------------------|
| General | 8-10 |
| Professional | 10-12 |
| Technical | 12-14 |

---

## 3. Tone (Weight: 15%)

### Definition
Appropriate voice, style, and emotional register for the target audience.

### Scoring Guide

| Score | Criteria |
|-------|----------|
| 90-100 | Perfect audience match |
| 80-89 | Appropriate with minor inconsistencies |
| 70-79 | Generally appropriate |
| 60-69 | Tone mismatches |
| < 60 | Wrong tone for audience |

### Tone Types

| Content Type | Expected Tone |
|--------------|---------------|
| Blog | Informative, engaging |
| Landing | Persuasive, confident |
| Product | Clear, benefit-focused |
| FAQ | Helpful, reassuring |
| Service | Professional, trustworthy |

### Checklist

- [ ] Consistent voice throughout
- [ ] Matches brand guidelines
- [ ] Appropriate formality level
- [ ] No jarring shifts

---

## 4. SEO (Weight: 15%)

### Definition
Search engine optimization elements and practices.

### Scoring Guide

| Score | Criteria |
|-------|----------|
| 90-100 | Fully optimized, all elements present |
| 80-89 | Well optimized, minor gaps |
| 70-79 | Basic optimization done |
| 60-69 | Significant SEO gaps |
| < 60 | Not optimized |

### Checklist

- [ ] Target keyword in title
- [ ] Keyword in first 100 words
- [ ] Keyword density 1-2%
- [ ] Meta description (150-160 chars)
- [ ] H1-H6 hierarchy correct
- [ ] Internal links (2-3 minimum)
- [ ] Image alt texts
- [ ] URL/slug optimized

---

## 5. Engagement (Weight: 15%)

### Definition
Reader retention, interest, and action-taking.

### Scoring Guide

| Score | Criteria |
|-------|----------|
| 90-100 | Compelling, hard to stop reading |
| 80-89 | Engaging with strong hook |
| 70-79 | Maintains interest |
| 60-69 | Loses reader in parts |
| < 60 | Boring or disengaging |

### Checklist

- [ ] Strong opening hook
- [ ] Compelling subheadings
- [ ] Scannable format
- [ ] Visual breaks
- [ ] Clear value proposition
- [ ] Effective CTA

### Engagement Elements

| Element | Purpose |
|---------|---------|
| Hook | Grab attention in first sentence |
| Questions | Involve reader |
| Stories | Create connection |
| Data | Provide credibility |
| CTA | Drive action |

---

## 6. E-E-A-T (Weight: 15%)

### Definition
Experience, Expertise, Authoritativeness, Trustworthiness signals.

### Scoring Guide

| Score | Criteria |
|-------|----------|
| 90-100 | Strong E-E-A-T throughout |
| 80-89 | Good signals present |
| 70-79 | Basic signals |
| 60-69 | Weak E-E-A-T |
| < 60 | No E-E-A-T signals |

### Checklist

- [ ] Author credentials visible
- [ ] First-hand experience shown
- [ ] Expert sources cited
- [ ] Authoritative references
- [ ] Trust signals (reviews, certifications)
- [ ] "Last updated" date
- [ ] Contact information

### E-E-A-T by Component

| Component | How to Demonstrate |
|-----------|-------------------|
| **Experience** | Real examples, case studies, "I did X" |
| **Expertise** | Credentials, technical depth |
| **Authoritativeness** | Citations, backlinks, recognition |
| **Trustworthiness** | HTTPS, transparency, accuracy |

## 8. Critical Issues (Auto-Fail)

Content automatically fails review if:

| Issue | Reason |
|-------|--------|
| Factual errors | Damages credibility |
| Plagiarism | Legal/ethical |
| Offensive content | Brand risk |
| Missing CTA | No business value |
| Broken structure | UX failure |

---

> **Principle:** Quality is measured, not assumed. Every criterion has clear, objective scoring guidelines.
