# Readability Optimization

> Making content easier to read and understand.

---

## 1. Readability Metrics

### Flesch Reading Ease

| Score | Level | Audience |
|-------|-------|----------|
| 90-100 | Very Easy | 5th grade |
| 80-89 | Easy | 6th grade |
| 70-79 | Fairly Easy | 7th grade |
| 60-69 | Standard | 8-9th grade |
| 50-59 | Fairly Difficult | High school |
| 30-49 | Difficult | College |
| 0-29 | Very Difficult | Graduate |

**Target:** 60-70 for most web content

### Key Factors

| Factor | Measurement |
|--------|-------------|
| Sentence length | Words per sentence |
| Word complexity | Syllables per word |
| Paragraph length | Sentences per paragraph |
| Structure | Headers, lists, whitespace |

---

## 2. Sentence Optimization

### Length Guidelines

| Type | Target | Max |
|------|--------|-----|
| Average | 15-20 words | 25 words |
| Variation | Mix short and long | - |
| Opening | Under 20 words | - |

### Techniques

**Too Long:**
> "The comprehensive digital marketing strategy that our team developed after extensive market research and competitor analysis has resulted in significant improvements across all key performance indicators."

**Better:**
> "Our team developed a comprehensive digital marketing strategy. The research included market analysis and competitor review. Results show significant improvement across all KPIs."

### Simplification Patterns

| Complex | Simple |
|---------|--------|
| In order to | To |
| Due to the fact that | Because |
| In the event that | If |
| At this point in time | Now |
| With regard to | About |

---

## 3. Word Choice

### Prefer Simple Words

| Complex | Simple |
|---------|--------|
| Utilize | Use |
| Implement | Start, do |
| Facilitate | Help |
| Leverage | Use |
| Optimize | Improve |
| Synergy | Teamwork |

### Avoid Jargon

Unless your audience expects it:
- Define technical terms
- Use common equivalents
- Explain acronyms first use

### Active Voice

| Passive (Avoid) | Active (Prefer) |
|-----------------|-----------------|
| The report was written by the team | The team wrote the report |
| Improvements were made | We improved |
| Results can be seen | You'll see results |

---

## 4. Paragraph Structure

### Guidelines

- **Length:** 2-4 sentences max
- **Focus:** One idea per paragraph
- **Flow:** Logical progression
- **Transition:** Connect paragraphs

### Visual Rhythm

```
[Short paragraph - 1-2 sentences]

[Medium paragraph - 3-4 sentences with detail]

[Short paragraph - key point]

• Bullet
• Points
• Here

[Closing paragraph]
```

---

## 5. Formatting for Scannability

### Elements

| Element | When to Use |
|---------|-------------|
| **Bold** | Key terms, important points |
| *Italic* | Emphasis, definitions |
| Bullet lists | Multiple related items |
| Numbered lists | Sequential steps |
| Tables | Comparisons, data |
| Quotes | Expert opinions, callouts |
| Headers | Section breaks |

### F-Pattern Reading

Users scan in F-pattern:
1. Top horizontal line (headline)
2. Second horizontal line (subhead)
3. Vertical line down left side

**Optimization:**
- Front-load important words
- Use meaningful subheads
- Left-align key content

---

## 6. Content Density

### White Space

- Margins around text
- Space between paragraphs
- Space around images
- Line height (1.5-1.7)

### Information per Section

- 1 main point per section
- 3-5 supporting details
- Break long sections into subsections

---

## 7. Readability Checklist

### Before Publishing

- [ ] Average sentence < 20 words
- [ ] Flesch score 60+ (for general audience)
- [ ] No paragraph > 4 sentences
- [ ] Headers every 300 words
- [ ] Simple word alternatives used
- [ ] Active voice dominant
- [ ] Adequate white space
- [ ] Key points scannable

### Tools

- Hemingway Editor
- Grammarly
- Yoast SEO readability
- Readable.com

## 8. Audience Adaptation

### By Expertise Level

| Audience | Approach |
|----------|----------|
| Expert | Technical terms OK, dense content |
| Professional | Industry terms, moderate depth |
| General | Plain language, more explanation |
| Beginner | Simple terms, step-by-step |

### By Content Type

| Type | Reading Level |
|------|---------------|
| News | 8th grade |
| Blog | 7-9th grade |
| Technical | Higher acceptable |
| Legal | Higher expected |
| Marketing | 6-8th grade |
