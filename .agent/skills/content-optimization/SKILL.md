
### Immediate Improvements

| Change | Impact |
|--------|--------|
| Add table of contents | Navigation, time on page |

| Add internal links | SEO, pageviews |

---

## 5. Content Audit Triggers

When to optimize:
- Traffic decline
- High bounce rate
- Low conversion
- Outdated information
- New keywords to target
- Competitor outranking

---

## 6. Optimization Checklist

### Readability
- [ ] Reading level appropriate for audience
- [ ] Sentences under 25 words average
- [ ] Paragraphs 2-4 sentences
- [ ] Active voice preferred
- [ ] Jargon minimized

### Structure
- [ ] Clear heading hierarchy
- [ ] Table of contents (for long content)
- [ ] Bullet points for lists
- [ ] Tables for comparisons
- [ ] Visual breaks (images, quotes)

### SEO
- [ ] Target keyword in title
- [ ] Keywords in first 100 words
- [ ] Meta description optimized
- [ ] Internal links added
- [ ] Image alt texts

### Engagement
- [ ] Compelling hook
- [ ] Clear value proposition
- [ ] Scannable format
- [ ] Strong CTA

---

> **Principle:** Optimization is iterative. Small improvements compound over time.
