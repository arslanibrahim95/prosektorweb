# SEO Enhancement

> Optimizing existing content for search visibility.

---

## 1. Content SEO Audit

### Quick Assessment

| Element | Check |
|---------|-------|
| Title | Keyword present, 50-60 chars |
| Meta description | Compelling, 150-160 chars |
| H1 | Single, contains keyword |
| URL | Short, keyword-included |
| First 100 words | Keyword present |
| Content length | Competitive with SERP |

---

## 2. Keyword Optimization

### Primary Keyword Placement

| Location | Priority |
|----------|----------|
| Title tag | Critical |
| H1 heading | Critical |
| URL | High |
| First paragraph | High |
| Meta description | High |
| At least one H2 | Medium |
| Image alt text | Medium |
| Throughout body | Low density |

### Keyword Density

- **Target:** 1-2% (natural use)
- **Avoid:** Keyword stuffing
- **Include:** Variations and synonyms

### LSI Keywords

Add related terms:
- Synonyms
- Related concepts
- Question variations
- Long-tail variations

---

## 3. Title Tag Optimization

### Formula

```
[Primary Keyword] - [Modifier] | [Brand]
```

### Modifiers

| Type | Examples |
|------|----------|
| Year | 2024, Updated |
| Quality | Best, Top, Ultimate |
| Format | Guide, Tutorial, Checklist |
| Specificity | For Beginners, Complete |

### Examples

```
Before: SEO Tips for Websites
After:  SEO Tips: 15 Proven Tactics for 2024 | YourBrand
```

### Rules
- Under 60 characters
- Keyword at front
- Compelling and accurate
- Unique per page

---

## 4. Meta Description

### Formula

```
[Hook/Question] [Value proposition] [CTA]
```

### Examples

```
Before:
"Learn about SEO and how to improve your website rankings."

After:
"Struggling to rank? Discover 15 proven SEO tactics that
increased our traffic by 300%. Start optimizing today."
```

### Rules
- 150-160 characters
- Include target keyword
- Compelling to click
- Accurate preview
- Include CTA

---

## 5. Heading Optimization

### Structure

```
H1: Primary Topic (1 only)
  H2: Main Section 1
    H3: Subsection
    H3: Subsection
  H2: Main Section 2
    H3: Subsection
  H2: FAQ Section
```

### Heading Tips

- Include keywords naturally
- Use questions (for featured snippets)
- Make descriptive, not generic
- Maintain logical hierarchy

### Examples

```
Before: Things to Know
After:  5 Essential SEO Ranking Factors for 2024

Before: How It Works
After:  How to Improve Your SEO Score in 7 Steps
```

## 6. Content Depth

### Comparison Analysis

Check competitors for:
- Word count
- Topics covered
- Media used
- Questions answered

### Content Gap Fill

Add sections for:
- Missing subtopics
- Unanswered questions
- Related concepts
- Updated information

### E-E-A-T Signals

| Signal | How to Add |
|--------|------------|
| Experience | Personal examples, case studies |
| Expertise | Author bio, credentials |
| Authority | Citations, backlinks |
| Trust | Sources, accuracy, transparency |

## 7. Internal Linking

### Strategy

| Link Type | Purpose |
|-----------|---------|
| Contextual | Related content in body |
| Navigation | Category/tag pages |
| Anchor text | Descriptive, keyword-rich |
| Distribution | 2-5 per 1000 words |

### Anchor Text Guidelines

```
Bad:  Click here, Read more, This article
Good: SEO best practices, content optimization guide
```

### Link Placement

- Within first few paragraphs
- At relevant mention points
- End of article (related posts)

---

## 8. Image Optimization

### File Names

```
Bad:  IMG_12345.jpg
Good: seo-optimization-checklist.jpg
```

### Alt Text

```
Bad:  image, photo, picture
Good: Checklist showing 10 SEO optimization steps
```

### Optimization Checklist

- [ ] Descriptive file names
- [ ] Alt text with keywords
- [ ] Compressed file size
- [ ] Appropriate dimensions
- [ ] WebP format where possible
- [ ] Lazy loading enabled

- Changed best practices
- New tools/methods

### Update Actions

| Action | SEO Impact |
|--------|------------|

| Add new sections | Content depth |

| Update screenshots | Accuracy |
| Add recent data | Authority |

## 10. Featured Snippet Optimization

### Target Formats

| Type | Format |
|------|--------|
| Paragraph | 40-60 word answer |
| List | Numbered/bullet steps |
| Table | Comparison data |
| Definition | "What is X" format |

### Optimization

- Use question as H2
- Answer immediately after
- Use proper formatting
- Be concise and direct

### Example

```markdown
## What is SEO?

SEO (Search Engine Optimization) is the practice of
optimizing websites to rank higher in search engine
results. It involves technical improvements, content
optimization, and building authority through backlinks.
```
