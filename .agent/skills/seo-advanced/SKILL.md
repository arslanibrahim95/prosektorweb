## 3. Schema.org Priority

| Schema Type | When to Use |
|-------------|-------------|
| `Organization` | Company/brand pages |
| `LocalBusiness` | Local businesses |
| `Article` | Blog posts, news |
| `Product` | E-commerce products |
| `FAQPage` | FAQ sections |
| `HowTo` | Tutorials, guides |
| `BreadcrumbList` | Navigation |
| `Person` | Author pages |

| Priority | Element | Impact |

| Medium | Hreflang | International SEO |
| Medium | Internal linking | PageRank flow |

## 5. Keyword Research Process

1. **Seed Keywords** - Core topic identification
2. **Expansion** - Related terms, questions
3. **Intent Analysis** - Informational vs transactional
4. **Competition** - Difficulty assessment
5. **Prioritization** - Value vs effort matrix
