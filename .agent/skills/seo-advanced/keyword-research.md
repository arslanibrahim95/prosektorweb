# Keyword Research

> Methodology for identifying and prioritizing target keywords.

---

## 1. Research Process

```
Seed Keywords → Expansion → Analysis → Prioritization → Mapping
```

---

## 2. Seed Keywords

### Sources
| Source | Method |
|--------|--------|
| Business goals | Core products/services |
| Competitors | Their ranking keywords |
| Customer language | Support tickets, reviews |
| Industry terms | Trade publications |

### Example
```
Business: OSGB (Occupational Health Services)
Seeds: iş sağlığı, işyeri hekimi, isg eğitimi, risk analizi
```

---

## 3. Keyword Expansion

### Methods
| Method | Tool/Technique |
|--------|----------------|
| Autocomplete | Google, YouTube search |
| Related searches | Bottom of SERP |
| People Also Ask | SERP feature |
| Question keywords | AnswerThePublic |
| Long-tail | Add modifiers |

### Modifiers
| Type | Examples |
|------|----------|
| Location | istanbul, ankara, online |
| Intent | nasıl, nedir, fiyat, en iyi |
| Comparison | vs, karşılaştırma, fark |
| Year | 2024, güncel |

---

## 4. Search Intent

### Intent Types
| Intent | Signal | Content Type |
|--------|--------|--------------|
| Informational | nedir, nasıl, neden | Blog, guide |
| Navigational | [brand] login | Landing page |
| Commercial | en iyi, karşılaştırma | Comparison |
| Transactional | satın al, fiyat | Product page |

### Matching Content
| Intent | Format |
|--------|--------|
| "X nedir" | Definition article |
| "X nasıl yapılır" | How-to guide |
| "X vs Y" | Comparison table |
| "X fiyat" | Pricing page |
| "X satın al" | Product page |

---

## 5. Keyword Metrics

### Key Metrics
| Metric | Description | Ideal |
|--------|-------------|-------|
| Search Volume | Monthly searches | Higher = more potential |
| Keyword Difficulty | Competition level | Lower = easier to rank |
| CPC | Ad cost | Higher = commercial value |
| SERP Features | Rich results present | Opportunity or barrier |

### Difficulty Levels
| Level | Score | Strategy |
|-------|-------|----------|
| Easy | 0-30 | Target quickly |
| Medium | 30-60 | Quality content wins |
| Hard | 60-80 | Need authority |
| Very Hard | 80+ | Long-term play |

---

## 6. Prioritization Matrix

```
              HIGH VOLUME
                  |
    Quick Wins    |    Big Bets
    (Easy + High) |    (Hard + High)
                  |
LOW DIFF ---------|--------- HIGH DIFF
                  |
    Fill Content  |    Avoid/Later
    (Easy + Low)  |    (Hard + Low)
                  |
              LOW VOLUME
```

### Scoring Formula
```
Priority = (Volume × 0.4) + (1 - Difficulty × 0.3) + (Intent Match × 0.3)
```

---

## 8. Content Gap Analysis

### Process
1. List competitor URLs
2. Extract their keywords
3. Compare with your keywords
4. Identify gaps
5. Prioritize opportunities

### Gap Types
| Gap | Action |
|-----|--------|
| Missing topics | Create new content |
| Thin content | Expand existing |
| Outdated content | Update and improve |
| No ranking | Optimize on-page |

---

## 9. Tracking

### Metrics to Track
- Keyword rankings
- Organic traffic by keyword
- Click-through rate
- Conversion by keyword

### Review Frequency
| Metric | Frequency |
|--------|-----------|
| Rankings | Weekly |
| Traffic | Monthly |
| Conversions | Monthly |
| Full audit | Quarterly |
